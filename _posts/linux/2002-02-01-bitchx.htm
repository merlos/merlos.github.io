Introducción

El BitchXes el cliente que suelo usar cuando me conecto a alguna red de IRC. Aquí iré incluyendo las cosas que me son útiles y que he ido aprendiendo poco a poco sobre este programa. El BitchX, en general, está bien documentado, en inglés, pero bien documentado. Un texto que es implrescindible leer si se va a utilizar con regularidad este cliente es el FAQ

BitchX

* En Debian, por defecto, al instalar el BitchX está activado el modo en el
  que se envían   los GLINEs y KLINES, son una lata, para quitarlos:
   /umode -isn 

* Añadir alias para dentificarse ante NiCK (IRC-hispano):
  /alias nick quote nick $0:$1
  Entonces,  identificarse habrá que teclear:
  /nick tunick tupass

* Alias Nueva Ventana:
  /alias wnh window new hide_others
  Entonces, para crear una nueva ventana:
  /wnh

* Teclas Rápidas:
  ESC-Número_ventana Para cambiar de ventana
  C-x Para Cambiar de canal dentro de una misma ventana
  F1 al F5 Muestran información como usuarios del canal, usuarios con @, 
           información sobre el estado de las conexiones DCC...

* Tanto para comenzar como finalizar privado: /q nick

* Salvar Opciones:
  /saverirc -all
  /save all
